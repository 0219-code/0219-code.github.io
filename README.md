# 春晚魔术计算器

这是一个结合标准计算器与科学计算器功能的网页应用，内置了“春晚魔术”效果。其核心逻辑基于以下 C++ 代码描述，并使用 JavaScript 实现。

```cpp
int a, b;
cin >> a >> b;
if (a >= 1000 && a <= 9999 && a == b) {
    cout << "当前时间";  // 实际输出月日时分
    system("exit");
} else {
    cout << a + b;
}
```

当用户连续输入两个相同的四位数并按下两次 `+` 后，计算器会显示当前时间的月日时分组合（例如 `2171720`），然后所有按钮（除 `C` 外）被禁用，模拟程序退出；否则显示两数之和，并可继续正常计算。

## 魔术效果说明

1. **正常计算**：可进行加减乘除、科学运算、内存读写等。
2. **触发魔术**：
   - 输入第一个四位数（如 `1234`），按下 **`+`**
   - 输入第二个四位数（必须与第一个完全相同，如 `1234`），再次按下 **`+`**
   - 屏幕显示第二个数，魔术预备状态激活
   - 按下**任意数字键**（0-9），屏幕内容立即替换为顶部面板显示的“月日时分”魔术数（如 `2171720`）
   - 随后所有按钮（除 `C` 外）变灰禁用，按 `C` 可重置并恢复正常。
3. **不满足条件**：若两数不都为四位数或不相等，按下第二个 `+` 后显示两数之和，可继续计算。

## 功能特性

- **标准运算**：加、减、乘、除、取模、幂运算
- **科学函数**：sin、cos、tan、ln、log、√、x²、1/x、exp、n!、rand、π、e
- **内存操作**：MC、MR、M+、MS
- **界面**：仿 Windows 计算器深色主题，按钮按压动画
- **实时时间**：顶部面板每秒更新当前时间和魔术预览数

## 自定义提示

- **修改魔术数字格式**：编辑 `getMagicFromTime()` 函数，调整月、日、时、分的拼接方式（如改为“年月日时分”）。
- **取消实时刷新**：注释掉 `setInterval(updateTimeDisplay, 1000);` 即可使时间仅加载时更新一次。
