# 🎩 春晚魔术计算器 · C++ 逻辑的 JavaScript 实现

这是一个结合了标准计算器与科学计算器功能的网页应用，最特别的是它内置了“春晚魔术”效果：**当用户连续输入两个相同的四位数并按下两次 `+` 后，计算器会显示当前时间的月日时分组合（例如 `2171720`），然后“冻结”所有按钮（仅 `C` 可解冻）**，完美模拟你提供的 C++ 逻辑。

> ✨ **核心逻辑来源**  
> 你曾用 C++ 描述了这个魔术：
> ```cpp
> int a, b;
> cin >> a >> b;
> if (a >= 1000 && a <= 9999 && a == b) {
>     cout << "当前时间";  // 实际输出月日时分
>     system("exit");
> } else {
>     cout << a + b;
> }
> ```
> 本项目用 **JavaScript + HTML/CSS** 完全实现了这一行为，并额外增加了科学计算、内存操作和键盘支持。

## 🪄 魔术效果演示

1. **正常计算**：你可以像使用普通计算器一样进行加减乘除、科学运算、内存读写。
2. **触发魔术**：
   - 输入第一个四位数（例如 `1234`），按下 **`+`**
   - 输入第二个四位数（必须与第一个**完全相同**，例如 `1234`），再次按下 **`+`**
   - 此时屏幕显示第二个数 `1234`，**魔术预备状态激活**
   - 接下来按下**任意数字键**（0-9），屏幕内容会**瞬间替换**为顶部面板显示的“月日时分”魔术数（如 `2171720`）
   - 随后所有按钮（除 `C` 外）变灰禁用，表示“程序退出”——只有按 `C` 才能重置并恢复正常。

3. **不满足条件时**：如果两个数不都是四位数，或者数值不相等，按下第二个 `+` 后会直接显示两数之和，并可以继续正常计算。

## 🧮 功能特性

- **标准运算**：加、减、乘、除、取模、幂运算
- **科学函数**：sin、cos、tan、ln、log、√、x²、1/x、exp、n!、rand、π、e
- **内存操作**：MC、MR、M+、MS（模拟经典计算器）
- **界面风格**：仿 Windows 计算器深色主题，按钮按压动画
- **实时时间显示**：顶部面板每秒更新当前时间和魔术预览数
